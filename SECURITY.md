# Security Policy

## Supported Versions

Security fixes are applied to the latest maintained version on the default branch.

## Reporting a Vulnerability

Please do not open public issues for security-sensitive problems.

Instead:

1. Share a private report with the maintainers
2. Include reproduction steps, impact, and any relevant logs
3. Redact tokens, secrets, private hosts, and customer data

We will review the report and respond as quickly as possible.

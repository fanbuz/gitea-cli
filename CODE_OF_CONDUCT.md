# Code of Conduct

## Our Pledge

We want this project to be a welcoming and respectful space for everyone.

## Our Standards

Examples of behavior that helps create a positive environment:

- being respectful and constructive
- assuming good intent
- giving actionable feedback
- focusing discussion on the work, not the person

Examples of unacceptable behavior:

- harassment or personal attacks
- insulting or demeaning language
- publishing private information without permission
- disruptive or bad-faith conduct

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, or pull requests that violate this Code of Conduct.

## Scope

This Code of Conduct applies in project spaces, including issues, pull requests, discussions, and related community channels.

## Reporting

Please report unacceptable behavior privately to the project maintainers through a non-public contact channel.
